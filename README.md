# SOS Plombier 66 — Site vitrine

Site vitrine **responsive** prêt pour **GitHub Pages**.

## Personnaliser
- Ouvre `script.js` et change `PHONE_DISPLAY` + `PHONE_TEL`
- Change l'email `contact@sosplomberie66.fr` dans `script.js`
- Modifie les textes dans `index.html`

## Publier sur GitHub Pages
1. Crée un repo GitHub
2. Upload tous les fichiers (ou `git push`)
3. Settings → Pages → Deploy from branch → `main` / `root`
